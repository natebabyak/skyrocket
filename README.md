## Getting Started

<!-- PREREQUISITES -->

### Prerequisites

- npm

  ```bash
  npm install npm@latest -g
  ```

<!-- INSTALLATION -->

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/natebabyak/skyrocket.git
   ```

2. Install the dependencies

   ```bash
   npm install
   ```

3. Start the local development server

   ```bash
   npm run dev
   ```
